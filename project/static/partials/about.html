About Southern Wholesome Education

Tools
    Bootstrap: Bootstraps is used as the front-end design framework.
    AWS Elastic Compute Cloud (EC2): BAll HTML and CSS files are hosted through an AWS EC2 instance.
    Flask: BFlask is used as the main web framework. The index.html file is routed through Flask. The HTTP requests to load in information into Angular is also requested through Flask.
    AngularJS: Angular is used to route in html partial pages based on links clicked.
    SQLAlchemy: SQLAlchemy was used instantiate our class our models.
